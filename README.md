# E-Plant Project

A Django + React E-shop developed with TDD.

## Tech Stack
- Backend: Django + DRF
- Frontend: React + TailwindCSS
- DB: PostgreSQL
- Cache: Redis (planned)
- Auth: JWT
- Testing: pytest, factoryboy

## Setup

### Prerequisites
- Docker & Docker Compose

### Running locally
```bash
docker-compose up --build
```
